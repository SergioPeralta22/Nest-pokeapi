# Ejecutar en modo desarrollo

1. Clonar el repositorio  
2. Ejecutar pnpm install
```
pnpm i 
```
3. Tener Nest CLI instalado
```
pnpm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```
